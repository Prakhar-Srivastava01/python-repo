# Python Practice Projects

This repository contains my Python practice projects created while learning Python programming.

## Projects

### 1. Calculator
A command-line calculator that performs basic arithmetic operations such as
- Addition
- Subtraction
- Multiplication
- Division
- Power

### 2. Number Guessing Game
A fun CLI game where the user tries to guess a randomly generated number within limited attempts.
Features:
- Random number generation
- Attempt limit
- User input validation

### 3. Smart To-Do Manager (CLI) ⭐

An advanced command-line task manager with real-world features.

Features:
- Add tasks with priority (High / Medium / Low)
- Mark tasks as completed
- Delete tasks
- Search tasks
- Automatic date & time tracking
- Data saved using JSON storage
